# Inspiria 5.0

Annual Techno-Management Symposium of TPC-PCE (Training & Placement Cell, Pillai College of Engineering).

## Features

- **Smooth Hyperspeed Background**: Advanced WebGL effects for a futuristic feel.
- **Responsive Design**: Optimized for mobile, tablet, and desktop views.
- **Event Timeline**: Interactive schedule of all symposium activities.
- **Gallery Marquee**: Smooth scrolling gallery showcasing past event highlights.
- **Premium Aesthetics**: Glassmorphism, gold shimmer effects, and custom typography.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS + Custom CSS
- **Icons**: FontAwesome & Lucide-react
- **Animations**: Framer Motion / Motion & GSAP

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run developmental server: `npm run dev`
4. Build for production: `npm run build`
